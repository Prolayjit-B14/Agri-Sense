# Platform Features

The AgriSense App provides a massive array of features tailor-made for modern agriculture.

## 📊 Dashboard & Analytics Hub
- **Real-Time Telemetry:** View live readouts of soil moisture, NPK values, and weather.
- **Chart Visualization:** Powerful line and area charts for historical data analysis using `Recharts`.
- **Soil Intelligence:** ML-driven evaluations of soil health quality.

## 📹 Security & Vision
- **Animal Detection:** Integrated ESP32-CAM feeds that detect wildlife intrusions.
- **Live Streams:** Fast WebSocket-based visual rendering directly inside the web and mobile app.

## 💧 Remote Irrigation & Solar Control
- **Manual Overrides:** Control water pumps from the `Irrigation Control` panel.
- **Solar Tracking:** Monitor solar cell outputs, battery capacity, and daylight analysis metrics.

## 🤖 AI & Advisory Systems
- **Crop Recommendation:** Suggests optimal crops based on current soil telemetry.
- **Pest Advisory:** Locates potential pest outbreaks tied to seasonal changes.
- **Market Insights:** Keep track of live market data for "Buy/Sell" advisories.

## 🌍 Platform Specifics
- **Geospatial Tracking:** Map Views detailing where physical nodes are placed on the farm.
- **Offline/Native capabilities:** Compiled as a native Android application using Capacitor with full tactile feedback and animations (`Framer Motion`).
